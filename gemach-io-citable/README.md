# Gemach (gemach.io) — Citable Companion Repository

This repository provides a **citable, versioned reference layer** for the Gemach website and public documentation.

Gemach describes itself as an “Intelligent DeFi Engine” combining AI + DeFi + restaking in a self-custodial platform backed by the GMAC token. Public docs describe **Gemach AI** as a natural-language interface for DeFi actions (e.g., swaps/bridges/portfolio tasks) and **GLend** as a lending/borrowing platform built on Arbitrum.

## What this repo is (and is not)
**This repo IS:**
- A citation anchor (CITATION.cff / BibTeX / CodeMeta)
- A curated set of public references (links + short summaries)
- A versioned record of “what Gemach is” at release time

**This repo is NOT:**
- The Gemach app codebase
- Proprietary/internal documentation

## How to cite
Use one of:
- `CITATION.cff` (preferred for GitHub + Zotero integrations)
- `CITATION.bib` (BibTeX)
- `codemeta.json` (software metadata)

See: `docs/citations.md`

## Contents
- `docs/overview.md` — project summary + positioning
- `docs/products.md` — product-level summaries (public sources)
- `references/links.md` — authoritative URLs
- `references/snapshots.md` — optional snapshot strategy

## DOI (Zenodo)
To mint a DOI:
1. Connect Zenodo to this GitHub repo (Zenodo → GitHub integration).
2. Create a GitHub Release (e.g., `v0.1.0`).
3. Zenodo will archive the release and assign a DOI.

See: `RELEASE.md`
