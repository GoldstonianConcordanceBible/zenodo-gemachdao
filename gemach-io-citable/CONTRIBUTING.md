# Contributing

Thanks for improving the citable reference layer.

## Allowed contributions
- Fixing links
- Updating public-source summaries with citations
- Adding new public documentation pages to `references/links.md`
- Improving citation metadata

## Not allowed
- Uploading proprietary code or private docs
- Copy/pasting large sections of third-party text

## Style
- Keep summaries short and factual
- Always prefer primary sources (official site/docs)
