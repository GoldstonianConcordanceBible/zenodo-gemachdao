# Versioning

This repository versions the *reference layer*, not the Gemach protocol/app.

- PATCH: link fixes, typos, minor wording changes
- MINOR: new public sources or new summary sections
- MAJOR: structural reorganizations or scope changes

Examples:
- v0.1.1 — fixed docs links
- v0.2.0 — added new product page references
- v1.0.0 — stable structure adopted
