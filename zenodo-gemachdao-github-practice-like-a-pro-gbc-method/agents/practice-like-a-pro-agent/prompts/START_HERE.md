# Start Here — Copy/Paste Prompts

## 1) Mode Select
"PLAP Agent: Put me in [PLAYER/PARENT/COACH/ENTREPRENEUR] mode.
My context: (age/level/goals/schedule). Build my next 7 days using the Greensboro Method."

## 2) Player Weekly Plan
"PLAP Agent (PLAYER): I have [X] days available, [Y] minutes/day, and games on [days].
Build a weekly plan with cage + field + strength + recovery.
Include a Rep Economy tracker and 3 focus cues."

## 3) Parent Structure Plan
"PLAP Agent (PARENT): My child is [age/level]. We struggle with [coachability/consistency/attitude].
Create a home structure plan (sleep, nutrition, routines), a car-ride script, and how to reinforce the coach."

## 4) Coach Session Script
"PLAP Agent (COACH): I’m running a [60/90] minute session for [age group].
Design stations, rep targets, key cues, and a correction script that builds men."

## 5) Entrepreneur Operating System
"PLAP Agent (ENTREPRENEUR): I run a training facility. Build policies for scheduling, cancellations, standards, and culture.
Include a 30-day systems improvement plan."
