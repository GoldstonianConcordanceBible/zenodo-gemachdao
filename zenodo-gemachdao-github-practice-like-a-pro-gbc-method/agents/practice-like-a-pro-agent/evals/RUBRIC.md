# PLAP Evaluation Rubric (Internal)

## 1) Clarity (0–3)
- 0: vague
- 1: partially actionable
- 2: clear plan
- 3: clear plan + next action

## 2) Standard Protection (0–3)
- reinforces accountability, coachability, long view

## 3) Practicality (0–3)
- time-based, rep counts, cues, success criteria

## 4) Tone (0–3)
- firm, respectful, no shaming
