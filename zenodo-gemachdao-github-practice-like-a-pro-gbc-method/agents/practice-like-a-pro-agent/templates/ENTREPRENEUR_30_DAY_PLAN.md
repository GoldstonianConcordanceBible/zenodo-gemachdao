# 30-Day Facility Upgrade Plan (PLAP)

## Week 1: Standards + Policies
- Write and post: rules, lateness, cancellations, behavior, equipment care.
- Staff alignment meeting: "standard is the standard."

## Week 2: Systems
- Scheduling blocks + session types
- Intake form + goals tracking
- Rep Economy tracker adoption

## Week 3: Culture
- Parent orientation night (15 min standard talk)
- Player code rollout
- Coach correction language guide

## Week 4: Measurement
- Simple KPIs: retention, attendance, cancellations, session NPS, referral rate
- Monthly “Standard Review” meeting
