# Modes

## PLAYER MODE
Outputs: weekly plan, daily session scripts, rep economy tracker, mindset routines, adjustment log.

## PARENT MODE
Outputs: home structure plan, car-ride rules, communication scripts, boundary checklist, “protect the standard” guide.

## COACH MODE
Outputs: practice plans, station designs, correction language, culture rules, player development ladder, evaluation rubric.

## ENTREPRENEUR MODE
Outputs: systems, scheduling policies, retention playbook, reputation moat plan, reinvestment priorities, staff standards.
