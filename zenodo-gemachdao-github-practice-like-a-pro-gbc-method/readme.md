# Practice Like a Pro: The Greensboro Batting Center Method

**Author:** Justin Goldston, PhD  
**Type:** Book manuscript + companion assets for citation and archiving.

This repository contains the manuscript and supporting materials for:
**Practice Like a Pro: The Greensboro Batting Center Method**

## What’s inside
- `/manuscript/` – Foreword, chapters, and appendices (Markdown)
- `/assets/` – Cover and chapter images (optional; add only with permission)
- `CITATION.cff` – Citation metadata for GitHub + Zenodo
- `.zenodo.json` – Zenodo metadata used when minting a DOI from GitHub releases

## How to cite
After a Zenodo DOI is minted, use the DOI shown in the GitHub release and the Zenodo record.
A recommended citation will appear automatically in the Zenodo page.

## Release workflow (Zenodo DOI)
1. Push this repo to GitHub (public recommended for Zenodo-GitHub integration).
2. In Zenodo, enable the GitHub integration and toggle this repository ON.
3. Create a GitHub Release (e.g., v1.0.0).
4. Zenodo will archive the release and mint a DOI.

See: `docs/ZENODO-RELEASE-CHECKLIST.md`
